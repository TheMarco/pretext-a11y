Accessibility in this demo

The visual layout uses Pretext's DOM-free text measurement to flow a long-form article across two columns with obstacle-aware reflow around rotating logos. All text positioning is computed in JavaScript and rendered as absolutely-positioned elements.

To make this content accessible, the full article text also exists as semantic HTML in this reading view. The visual stage is hidden from assistive technologies using aria-hidden. Screen readers, find-in-page, browser reader mode, and zoom all work against the semantic article, not the visual fragments. A view toggle lets sighted users switch between the visual demo and the readable article.
